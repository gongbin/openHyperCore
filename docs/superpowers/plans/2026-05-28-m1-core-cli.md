# OpenHyperCore M1 Core CLI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the first testable M1 slice: Scene Graph IR, JSX runtime primitives, deterministic frame evaluation, SVG single-frame renderer, and a local `openhyper probe/still` CLI.

**Architecture:** Keep the engine split into small packages that mirror `Plans.md`: `core` owns serializable IR and timeline evaluation, `jsx-runtime` converts JSX factory calls into IR nodes, `renderer-svg` provides a dependency-free still-frame backend until CanvasKit is integrated, and `cli` wires local composition loading to probe/still commands. The SVG backend is intentionally temporary but uses the same resolved-frame boundary expected by the future Skia backend.

**Tech Stack:** Node 24 native TypeScript stripping, node:test, pnpm workspace, dependency-free TypeScript modules.

---

## File Structure

- `package.json`: workspace scripts for tests and CLI smoke runs.
- `pnpm-workspace.yaml`: package discovery.
- `packages/core/src/types.ts`: IR and resolved-frame type definitions.
- `packages/core/src/composition.ts`: composition factory and validation.
- `packages/core/src/scheduler.ts`: frame count/time conversion and layer filtering.
- `packages/core/src/animation.ts`: keyframe interpolation.
- `packages/core/src/index.ts`: package exports.
- `packages/core/test/core.test.ts`: core behavior tests.
- `packages/jsx-runtime/src/jsx-runtime.ts`: JSX factory functions and layer constructors.
- `packages/jsx-runtime/src/index.ts`: package exports.
- `packages/jsx-runtime/test/jsx-runtime.test.ts`: JSX conversion tests.
- `packages/renderer-svg/src/render-svg.ts`: resolved frame to SVG renderer.
- `packages/renderer-svg/src/index.ts`: package exports.
- `packages/renderer-svg/test/render-svg.test.ts`: still renderer tests.
- `packages/cli/src/index.ts`: CLI parser, module loading, probe/still commands.
- `packages/cli/test/cli.test.ts`: CLI smoke tests.
- `examples/simple-video.ts`: sample command-style composition.
